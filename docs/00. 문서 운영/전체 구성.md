# 전체 구성

## 목적

이 문서는 풀스택 부트캠프 15기의 커리큘럼을 한눈에 확인하기 위한 기준 문서입니다.

## 정리 방식

과정 순서와 제목은 `references/raw/notion`의 폴더명을 기준으로 유지합니다. 각 과정의 내부 자료는 아래 세 가지 기준으로만 개선합니다.

- 퀴즈: 난이도 조정, 자세한 해설, 문제은행 추가
- 문장: 교과서형 설명으로 정리
- 코드와 실습: `references/code/*`의 실습 코드를 바탕으로 최신 라이브러리 기준에 맞게 개선
