# 중요도 별표 기준

이 문서는 `courses/*/강의자료.md`의 제목에 붙는 중요도 별표(`⭐`)를 판단하는 기준입니다.

별표는 학생이 복습 우선순위를 판단하도록 돕는 표시입니다. 강사가 중요하다고 느끼는 모든 내용에 붙이는 표시가 아니라, 이후 학습과 실습 성공에 미치는 영향을 기준으로 제한해서 사용합니다.

## 적용 대상

별표는 학생용 개념 본문 제목에만 붙입니다.

붙일 수 있는 제목:

- 핵심 개념 설명
- 필수 문법 설명
- 이후 실습에서 반복해서 사용하는 개념
- 자주 틀리는 개념을 정리하는 제목

붙이지 않는 제목:

- `#스크립트`
- `#실습`
- `#퀴즈`
- `#요약`
- `#Video`, `#비디오`
- 설치, 링크, 참고 자료, 읽어 보기, 꿀팁
- 단순 코드 따라 하기 단계
- 같은 개념을 반복해서 다루는 하위 제목

## 별 개수 기준

### 별 없음

다음 중 하나에 해당하면 별표를 붙이지 않습니다.

- 단순 안내, 링크, 설치, 환경 설정입니다.
- 강의 진행을 위한 작업 순서입니다.
- 해당 시점에는 읽고 넘어가도 이후 학습이 막히지 않습니다.
- 필요한 경우 검색하거나 문서를 보고 해결할 수 있습니다.
- 이미 별표가 붙은 상위 개념의 하위 예시입니다.

### `⭐`

알아두면 도움이 되지만, 이후 학습의 핵심 전제는 아닌 내용입니다.

- 특수 문법 또는 예외입니다.
- 자주 보이지만 외워야 할 필요는 낮습니다.
- 실습에서 한두 번 사용되지만, 틀려도 쉽게 고칠 수 있습니다.
- 핵심 개념을 이해한 뒤 보충으로 다루면 좋은 내용입니다.

### `⭐⭐`

실습에서 반복해서 사용되고, 학생이 자주 헷갈리는 보조 핵심 개념입니다.

- 같은 과정 안에서 여러 번 반복됩니다.
- 실습 결과에 직접 영향을 줍니다.
- 틀리면 화면 결과가 달라지지만, 전체 진도가 막히지는 않습니다.
- 다음 과정에서 다시 등장할 가능성이 있습니다.

### `⭐⭐⭐`

이후 진도를 따라가기 위해 반드시 이해해야 하는 핵심 구조나 필수 문법입니다.

- 다음 여러 강의와 실습의 전제가 됩니다.
- 이해하지 못하면 이후 설명을 따라가기 어렵습니다.
- 실습 실패의 주요 원인이 될 수 있습니다.
- 다른 과정에서도 반복해서 사용됩니다.
- 단순 암기보다 개념 구조를 이해해야 하는 내용입니다.

## 판정 절차

제목마다 아래 질문을 순서대로 확인합니다.

1. 이 내용이 이후 3개 이상의 강의나 실습에서 반복해서 사용되는가?
2. 이 내용을 모르면 이후 설명이나 실습이 막히는가?
3. 학생이 자주 틀리거나 오해하는 지점인가?
4. 단순 검색으로 해결하기보다 개념 이해가 필요한가?
5. 설치, 링크, 실습, 퀴즈, 요약, 스크립트가 아닌 학생용 개념 본문인가?

판정:

- 1개 이하만 해당: 별 없음
- 2개 해당: `⭐`
- 3개 해당: `⭐⭐`
- 4개 이상 해당: `⭐⭐⭐`
- 5번에 해당하지 않으면 다른 조건을 만족해도 별표를 붙이지 않습니다.

## 개수 제한

별표가 너무 많으면 학생이 무엇을 우선해야 하는지 알기 어렵습니다.

- 한 `##` 단원 안에서 `⭐⭐⭐`는 2개를 넘기지 않는 것을 원칙으로 합니다.
- 한 `##` 단원 안에서 별표가 붙은 제목은 전체 `###` 제목의 30%를 넘기지 않는 것을 원칙으로 합니다.
- 같은 개념이 여러 하위 제목으로 나뉘어 있으면 첫 번째 핵심 제목에만 별표를 붙입니다.
- 기존 원문에 별표가 많으면 새 기준에 따라 줄이는 쪽으로 검토합니다.

## 예시

웹 퍼블리싱 초반 과정 기준 예시입니다.

- `⭐⭐⭐`: HTML 문서 구조, 핵심 태그, CSS 박스 모델, 크기와 여백처럼 이후 실습의 전제가 되는 내용
- `⭐⭐`: 문자 인코딩, 글꼴, 정렬처럼 자주 쓰이고 실습 결과에 영향을 주는 내용
- `⭐`: 줄 바꿈, 특수 문자 표시처럼 알아두면 좋지만 검색으로도 쉽게 보완할 수 있는 내용
- 별 없음: 프로그램 설치, VS Code 설정, 단순 실습 링크, 퀴즈, 요약, 개발자 도구 소개

## 검토 결과 기록

별표를 조정했으면 필요할 때만 아래처럼 짧게 남깁니다.

```markdown
- 별표 조정 대상:
- 조정 기준:
- `⭐⭐⭐` 개수:
- 별표를 제거한 이유:
- 사용자 확인 필요:
```
