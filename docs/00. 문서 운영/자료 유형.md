# 자료 유형

부트캠프 자료는 목적에 따라 문서 유형을 나누어 관리합니다.

## 목차

`목차.md`는 과정의 전체 흐름을 보여주는 문서입니다.

포함할 내용:

- 섹션 목록
- 강의 목록
- 강의, 실습, 퀴즈 여부
- 선수 지식 또는 준비물

## 교재

`교재/`는 개념 설명 중심의 자료입니다.

권장 구성:

- 개념 소개
- 필요한 배경 지식
- 코드 예시
- 동작 원리
- 정리

## 대본

`대본/`은 영상 촬영 또는 라이브 강의 진행을 위한 말하기 문서입니다.

권장 구성:

- 도입
- 이번 시간 목표
- 설명 흐름
- 실습 진행 멘트
- 마무리

## 실습

`실습/`은 학생이 직접 작업하는 단계별 문서입니다.

권장 구성:

- 실습 목표
- 사전 준비
- 체크리스트
- 검증 방법
- 자주 발생하는 오류

## 강의

`강의/`는 짧은 강의 단위 자료를 관리합니다.

권장 구성:

```text
강의/
  1-01. 강의명/
    학생용.md
    quiz.md
```

## 퀴즈

`quiz.md`는 강의 이해도를 확인하기 위한 문서입니다.

권장 구성:

- 문제
- 보기 또는 코드
- 정답
- 해설

정답을 학생에게 바로 공개하지 않을 경우, 별도 파일로 분리할 수 있습니다.
