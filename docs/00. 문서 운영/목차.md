# 00. 문서 운영

15기 강의 자료를 만들 때의 최소 운영 기준입니다.

## 문서

### 01. 15기 강의자료 개선 기준

- 11기 원문을 15기 자료로 개선하는 기준
- 퀴즈 난이도, 해설, 문제은행 보강 기준
- 교과서형 문장으로 다듬는 기준
- `references/code/*`의 실습 코드를 바탕으로 코드와 실습을 개선하는 기준
- `courses/NN.제목/강의자료.md` 형태의 과정별 단일 문서 관리 기준
- Notion 업로드 직전에 `$notion-formatter`로 `_notion.md`를 생성하는 기준

### 02. 전체 구성

- 01부터 38까지의 기준 과정 목록
- 01~10 과정의 현재 문서 위치
- 아직 15기 문서를 만들지 않은 과정의 원본 위치

### 03. 01 기준 샘플

- `01. 프로그래밍 시작하기`의 로컬 원본과 `_notion.md` 생성 기준
- Notion 읽기 전용 동기화 절차
- formatter와 publisher dry-run 검증 절차
- 퀴즈 시간, 태그, 이미지, 파일 링크, 강사용/학생용 영역 점검 결과

불필요한 공통 템플릿은 두지 않습니다. 과정별 자료는 원본 흐름을 유지하면서 필요한 부분만 수정합니다.
