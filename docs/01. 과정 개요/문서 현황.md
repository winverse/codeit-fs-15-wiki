# 문서 현황

## 현재 상태

- 저장소 루트에는 `README.md`가 있습니다.
- 수업 자료는 `docs/` 아래에서 관리합니다.
- 실제 강의 내용은 아직 확정본으로 옮기지 않았습니다.
- 기존 JavaScript 과정 자료 저장소의 Markdown 구조만 먼저 참고했습니다.

## 이후 정리할 내용

- 15기 실제 커리큘럼
- 주차별 또는 과정별 수업 흐름
- 강의별 학생용 자료
- 실습 가이드
- 퀴즈와 해설
- 프로젝트 요구사항
- 환경 설정 문서

## 업데이트 원칙

- 변경된 커리큘럼은 `02. 커리큘럼`에서 먼저 정리합니다.
- 강의별 본문은 과정 디렉터리가 생긴 뒤 `교재/`, `대본/`, `실습/`, `강의/`로 나눕니다.
- 버전 의존성이 있는 내용은 작성일 또는 검증일을 함께 기록합니다.
