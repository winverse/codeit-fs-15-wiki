# 운영 메모

부트캠프 운영 과정에서 반복해서 확인해야 하는 내용을 기록하는 문서입니다.

## 자료 관리

- 수업 전에 학생용 자료와 실습 자료의 링크를 확인합니다.
- 실습 자료는 명령어, 파일명, 예상 결과가 실제와 맞는지 확인합니다.
- 오래된 패키지 버전이나 CLI 옵션은 배포 전에 다시 검증합니다.

## 수업 후 업데이트

- 수업 중 발견한 오류를 해당 문서에 반영합니다.
- 질문이 반복된 내용은 FAQ 또는 보충 설명으로 분리합니다.
- 실습 실패 사례는 `자주 발생하는 오류` 섹션에 누적합니다.
