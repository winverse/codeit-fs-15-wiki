# codeit-fs-15 docs

이 디렉터리는 코드잇 풀스택 부트캠프 15기 자료를 모아두는 문서 루트입니다.

현재는 11기 자료를 바탕으로 15기 강의 자료를 만들기 위한 문서 골격과 작성 기준을 정리한 상태입니다.

## 구성

- [00. 문서 운영](./00.%20문서%20운영/목차.md): 문서 작성 규칙, 디렉터리 규칙, 자료 유형 정의
- [01. 과정 개요](./01.%20과정%20개요/목차.md): 부트캠프 문서의 목적과 현재 정리 상태
- [02. 커리큘럼](./02.%20커리큘럼/목차.md): 전체 커리큘럼을 정리할 자리
- [03. 자료 템플릿](./03.%20자료%20템플릿/목차.md): 강의, 퀴즈, 실습, 교재, 대본 템플릿

## 현재 작성 기준

- 15기 강의 순서와 과정 제목은 `references/raw/notion`의 `NN.제목` 폴더명을 기준으로 유지합니다.
- 내부 설명, 예제, 실습, 코드, 버전 안내만 15기 기준으로 개선합니다.
- 11기 코드 참고 저장소는 `https://github.com/winverse/codeit-fs.git`의 `codeit-fs-11/`입니다.
- 세부 작업 기준은 [11기 자료 기반 15기 강의자료 제작 기준](./00.%20문서%20운영/11기%20자료%20기반%2015기%20강의자료%20제작%20기준.md)을 따릅니다.

## 참고한 작성 패턴

기존 자료 저장소에서는 다음 패턴을 사용합니다.

- 참고 저장소: https://github.com/winverse/codeit-fs-javascript

```text
NN. 과정명/
  목차.md
  강의/
    S-LL. 강의명/
      학생용.md
      quiz.md
  교재/
    1_교재.md
  대본/
    1-01.강의명.md
  실습/
    1-01_강의명_실습.md
```

이 저장소에서는 모든 자료를 `docs/` 아래에 모으고, 과정별 세부 내용은 이후 확정되는 커리큘럼에 맞춰 추가합니다.
