# codeit-fs-15 docs

이 디렉터리는 이 레포의 운영 문서와 제작 기준을 모아두는 문서 루트입니다.

실제 15기 강의자료 본문은 `courses/` 아래에 둡니다. `docs/`에는 강의자료 제작 기준, 커리큘럼 진행 상태, 참고자료 정리 원칙만 둡니다.

## 구성

- [00. 문서 운영](./00.%20문서%20운영/목차.md): 15기 강의자료 개선 기준
- [01 기준 샘플](./00.%20문서%20운영/01%20기준%20샘플.md): `01. 프로그래밍 시작하기` 동기화, formatter, 품질 점검 기준
- [전체 구성](./00.%20문서%20운영/전체%20구성.md): 기준 과정 목록

## 현재 작성 기준

- 15기 강의 순서와 과정 제목은 `references/raw/notion`의 `NN.제목` 폴더명을 기준으로 유지합니다.
- 과정별 강의자료는 `courses/NN.제목/강의자료.md`처럼 과정 폴더 바로 아래에 둡니다.
- 내부 내용은 퀴즈, 문장, 코드와 실습 중심으로 개선합니다.
- `#스크립트`는 강사용 수업 진행 대본으로 보고, 학생용 안내는 `#실습`, `#퀴즈`, 링크 안내, 완료 기준에 반영합니다.
- 단순 링크만 있는 퀴즈 섹션은 그대로 두고, 링크 내용을 확인한 경우에만 문제은행형 퀴즈를 추가합니다.
- 실습 코드는 `references/code/*`를 기준으로 확인합니다.
- Notion 업로드용 `_notion.md`는 원본 Markdown이 확정된 뒤 `$notion-formatter`로 생성합니다.
- 세부 작업 기준은 [15기 강의자료 개선 기준](./00.%20문서%20운영/15기%20강의자료%20개선%20기준.md)을 따릅니다.

## 정리 원칙

- 공통 템플릿을 채우기 위한 문서는 만들지 않습니다.
- 실제 과정 문서는 `courses/`에만 추가합니다.
- `docs/`에는 원본을 그대로 옮기지 않고 운영 기준과 진행 기록만 남깁니다.
- Notion 변환본은 업로드용 파생 파일이므로 원본 문서와 섞어서 관리하지 않습니다.
