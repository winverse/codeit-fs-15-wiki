# 전체 구성

## 목적

이 문서는 풀스택 부트캠프 15기의 커리큘럼을 한눈에 확인하기 위한 기준 문서입니다.

## 정리 방식

과정 순서와 제목은 `references/raw/notion`의 폴더명을 기준으로 유지합니다. 15기 자료 작성 시에는 아래 항목을 과정별로 채웁니다.

```markdown
## NN. 과정명

### 목표

- 이 과정에서 학생이 할 수 있어야 하는 일

### 핵심 주제

- 주요 개념
- 주요 도구
- 주요 실습

### 산출물

- 완성 코드
- 프로젝트
- 제출 자료
```
