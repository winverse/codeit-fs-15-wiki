```markdown
## NN. 과정명

### 개요

과정의 역할과 학습 범위를 작성합니다.

### 선수 지식

- 필요한 배경 지식

### 학습 목표

- 학생이 설명할 수 있어야 하는 개념
- 학생이 구현할 수 있어야 하는 기능

### 강의 구성

| 번호 | 제목 | 유형 | 자료 |
| :--- | :--- | :--- | :--- |
| 1-01 | 제목 | 강의 | 링크 |

### 실습

- 실습명
- 결과물

### 검증 기준

- 체크 기준
- 제출 기준
```
