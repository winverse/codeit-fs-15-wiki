# codeit-fs-15-wiki

코드잇 풀스택 부트캠프 15기 수업 자료와 운영 문서를 정리하는 저장소입니다.

## 문서

- [docs](./docs): 15기 강의자료 개선 기준과 커리큘럼 기준 목록
- [references](./references): 원본 Markdown, PDF, 이전 기수 코드, 실습 코드 참고자료
- [15기 강의자료 개선 기준](./docs/00.%20문서%20운영/15기%20강의자료%20개선%20기준.md): 11기 자료와 `references/code/*`를 바탕으로 15기 자료를 개선하는 기준

## 개선 방향

- 과정 순서와 제목은 유지합니다.
- 퀴즈는 난이도, 해설, 문제은행 중심으로 보강합니다.
- 문장은 교과서형 설명으로 다듬습니다.
- 코드는 `references/code/*`의 실습 코드를 바탕으로 최신 라이브러리 기준에 맞게 개선합니다.
- Notion 업로드용 Markdown은 원본 문서가 확정된 뒤 `$notion-formatter`로 별도 생성합니다.
- 공통 템플릿을 강제하지 않고, 필요한 과정 문서만 추가합니다.
