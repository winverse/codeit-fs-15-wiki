# codeit-fs-15-wiki

코드잇 풀스택 부트캠프 15기 수업 자료와 운영 문서를 정리하는 저장소입니다.

## 문서

- [docs](./docs): 수업 자료, 커리큘럼, 문서 작성 규칙
- [references](./references): PDF, Markdown, 이전 기수 코드 등 참고자료
- [15기 강의자료 제작 기준](./docs/00.%20문서%20운영/11기%20자료%20기반%2015기%20강의자료%20제작%20기준.md): 11기 자료와 로컬 참고 코드를 바탕으로 15기 문서를 만드는 기준

## 참고 구조

문서 구조는 기존 JavaScript 과정 자료 저장소의 Markdown 구성 방식을 참고합니다.

- 참고 저장소: https://github.com/winverse/codeit-fs-javascript

- 과정별 디렉터리: `NN. 과정명`
- 과정 목차: `목차.md`
- 자료 유형: `교재`, `대본`, `실습`, `퀴즈`, `자료`, `완성본`
- 강의 자료는 대상별로 나누지 않는 단일 문서 기준으로 작성
