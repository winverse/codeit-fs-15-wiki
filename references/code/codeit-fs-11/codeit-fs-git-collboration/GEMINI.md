# 정리 규칙:

아래 강의 내용을 노션 스타일 핵심 요약 형태로 정리해줘.

- 각 항목은 마크다운 구분선과 리스트로 구분
- 예시는 코드블록으로 표시
- 옵션, 주의사항, 용어설명 따로 정리
- 맨 끝에 . 제거
- 이모티콘, 장식 빼고 명확하게!

# 규칙

- 항상 `목차.md`를 읽고서 실행 할 것
- `목차`에 의해서 이전에 생성된 .md 파일들이 있다면 뒤에 있는 내용과 이어지도록 구성하기
- 이것은 git을 모르는 학생들에게 가르치지 위해서 만드는 학습지
- 퀴즈 형식의 파일은 문제 -> 정답 및 해설 -> 문제 순서로 구성한다
