# codeit-fs-jest-practice 안내

이 폴더는 교재 각 장 마지막에 있는 실습을 할 수 있는 장별 코드 경로입니다.

- `01-jest-ts-esm-practice`: 1장 마지막 실습 코드
- `02-matchers-and-mocks-practice`: 2장 마지막 실습 코드
- `03-jest-basic-practice`: 3장 마지막 실습 코드
- `04-express-foundation-practice`: 4장 마지막 실습 코드
- `05-express-unit-practice`: 5장 마지막 실습 코드
- `06-supertest-e2e-practice`: 6장 마지막 실습 코드
- `08-final-practice`: 8장 마지막 종합 실습 코드
- 교재 본문을 다 따라간 뒤, 장 끝 실습을 진행할 때 이 경로를 사용합니다.
- 7장은 별도 실습이 없습니다.
- 다만 7장은 별도 `practice` 폴더 없이, 해당 장 코드 안에서 테스트 환경과 coverage 점검 흐름을 다시 확인하는 방식으로 운영합니다.
- 각 `practice` 경로는 실제 디렉터리로 구성되어 있습니다.
- 실습 코드는 대응하는 `starter` 기준과 같은 시작 상태를 담고 있지만, 저장소에서는 독립 폴더로 관리합니다.
