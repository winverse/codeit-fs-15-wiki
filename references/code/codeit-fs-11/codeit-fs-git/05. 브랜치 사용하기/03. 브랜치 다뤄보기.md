### 03. 브랜치 다뤄보기

이번 시간에는 브랜치를 조회하고, 생성, 이동, 삭제하는 기본적인 명령어들을 알아보겠습니다.

---

### 브랜치 목록 보기

- `git branch` 명령어를 사용하면 로컬 저장소에 있는 모든 브랜치의 목록을 볼 수 있습니다.
- 목록에서 `*` 기호가 붙어있는 브랜치가 현재 내가 작업하고 있는 브랜치입니다.

```bash
git branch
```
```
  main
* premium
```

---

### 브랜치 생성 및 이동

- **생성과 동시에 이동**: `git switch -c [새 브랜치 이름]`
- **다른 브랜치로 이동**: `git switch [이동할 브랜치 이름]`

---

### 브랜치 삭제하기

- `git branch -d [삭제할 브랜치 이름]` 명령어로 브랜치를 삭제할 수 있습니다. (`-d`는 delete를 의미)

#### 주의사항

- **현재 작업 중인 브랜치는 삭제할 수 없습니다.**
- 브랜치를 삭제하려면, 먼저 `git switch` 명령어로 다른 브랜치로 이동한 후에 삭제를 시도해야 합니다.

---

### 예시: `test` 브랜치 생성 후 삭제하기

```bash
# 1. 'test' 브랜치를 새로 만들고 이동합니다
git switch -c test

# 2. 현재 위치가 'test'이므로 삭제 시 에러가 발생합니다
git branch -d test
# error: Cannot delete branch 'test' checked out at '...'

# 3. 다른 브랜치('main')로 먼저 이동합니다
git switch main

# 4. 이제 'test' 브랜치를 안전하게 삭제할 수 있습니다
git branch -d test
# Deleted branch test (...).

# 5. 브랜치 목록을 다시 확인하여 삭제되었는지 봅니다
git branch
```
