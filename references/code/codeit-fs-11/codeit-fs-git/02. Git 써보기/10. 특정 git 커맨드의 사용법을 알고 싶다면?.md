# 10. 특정 git 커맨드의 사용법을 알고 싶다면?

Git에는 수많은 명령어가 있고, 각 명령어마다 다양한 옵션이 존재합니다. 모든 것을 외우는 것은 불가능에 가깝습니다. 다행히 Git은 자체적으로 상세한 도움말을 볼 수 있는 기능을 제공합니다.

---
### `git help` 또는 `--help` 사용하기

궁금한 명령어 뒤에 `--help`를 붙이거나 `git help <명령어>`를 입력하면, 해당 명령어의 공식 메뉴얼(설명서)을 볼 수 있습니다.

- **사용법**
  ```bash
  # 'add' 명령어의 도움말을 보고 싶을 때
  git add --help
  ```
  또는
  ```bash
  git help add
  ```

- **실행 결과**
  - 위 명령어를 실행하면, `git add`가 어떤 역할을 하는지, 어떤 옵션들을 사용할 수 있는지, 그리고 사용 예시는 무엇인지 등이 담긴 상세한 설명서가 웹 브라우저나 터미널에 나타납니다.

---
### 왜 중요할까요?

개발을 하다 보면 처음 보는 명령어나 옵션을 사용해야 할 때가 많습니다. 그럴 때마다 인터넷 검색에만 의존하기보다, Git이 제공하는 공식 도움말을 통해 가장 정확하고 신뢰도 높은 정보를 얻는 습관을 들이는 것이 좋습니다.
