# 02. repository 만들기

### **Repository란?**

- **Repository의 개념**
    - Git으로 버전 관리를 수행하는 폴더
    - 로컬 저장소(Local Repository)라고 부름

---

- **Repository 생성 방법**
    1. 버전 관리를 시작할 폴더로 이동 (폴더 생성후에)
    2. `git init` 명령어 실행으로 저장소 초기화

---

- **예시**
    
    ```bash
    mkdir mathTool
    cd mathTool
    git init
    ```
    
    - 새 폴더 생성 후 Git 저장소를 초기화함

---

## 주의사항

- `git init` 실행 시 `.git` 숨김 폴더가 생성됨
- `.git`에는 버전 관리에 필요한 모든 정보가 저장됨
- `.git` 폴더 삭제 시 프로젝트의 모든 Git 이력이 사라짐

---

## 용어설명

- **`.git 디렉토리`:** Git의 내부 데이터베이스로, 커밋 기록, 브랜치, 설정 정보 등이 저장되는 핵심 폴더
