# 04. calculator.js 파일에 작성했던 코드 설명

### **파일 개요**

- `calculator.js`는 덧셈과 뺄셈을 수행하는 두 개의 함수로 구성된 간단한 계산기 프로그램

---

- **함수(Function)의 개념**
    - 특정 동작을 수행하도록 작성된 코드의 묶음
    - 함수 이름을 호출하는 것만으로 재사용 가능

---

- **코드 설명**
    - **add 함수**
        - 두 숫자 `a`, `b`를 받아 합계를 반환
        - `return` 명령어를 사용해 결과를 반환
        
        ```jsx
        function add(a, b) {
          return a + b;
        }
        ```
        
    - **subtract 함수**
        - 두 숫자 `a`, `b`를 받아 차를 반환
        
        ```jsx
        function subtract(a, b) {
          return a - b;
        }
        ```
        

---

## 용어설명

- **function**
    
    함수를 정의하는 명령어
    
- **a, b**
    
    함수 실행 시 입력되는 값, 즉 인자(argument)
    
- **return**
    
    함수 수행 결과를 호출한 곳으로 돌려주는 명령어
