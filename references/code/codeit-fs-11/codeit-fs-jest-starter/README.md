# codeit-fs-jest-starter

이 디렉터리는 교재를 따라서 입력할 수 있도록 준비된 장별 시작 환경을 둡니다.

- `01-jest-ts-esm`: 1장 본편 시작 환경
- `02-matchers-and-mocks`: 2장 따라쓰기 시작 환경
- `03-jest-basic`: 3장 따라쓰기 시작 환경
- `04-express-foundation`: 4장 따라쓰기 시작 환경
- `05-express-unit`: 5장 따라쓰기 시작 환경
- `06-supertest-e2e`: 6장 따라쓰기 시작 환경
- `07-prisma-client-testing`: 7장 따라쓰기 시작 환경
- `08-final`: 8장 따라쓰기 시작 환경
- 각 폴더에는 교재를 따라 입력할 수 있도록 필요한 파일과 일부 스켈레톤을 미리 준비합니다.
- 1장 본편은 `01-jest-ts-esm`에서 시작합니다.
- 교재를 따라 입력한 뒤의 완성 상태는 `codeit-fs-jest/codeit-fs-jest-completed/`에서 확인합니다.
- 각 장 마지막 실습은 `codeit-fs-jest/codeit-fs-jest-practice/` 경로로 안내합니다.
- 현재 `practice` 경로는 일부 `starter` 폴더를 재사용하지만, 문서상 역할은 별도로 구분합니다.
